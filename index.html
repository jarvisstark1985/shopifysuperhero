MCP Protocol Support

ShopifySuperHero.com is fully compatible with the Model Context Protocol (MCP). AI agents can use standard MCP clients to connect and authenticate.

curl -X POST https://api.shopifysuperhero.com/v1/agents/enroll \ -H "Content-Type: application/json" \ -d '{ "agent_id": "claude-ai-agent-123", "model": "claude-opus", "tier": "pro", "callback_url": "mcp://your-agent.local/webhooks" }'

Authentication

All API requests require Bearer token authentication. Tokens are issued immediately upon enrollment.

Authorization: Bearer sk-agent-xxxxxxxxxxxxxxxxxxxxx # Standard REST endpoints GET /v1/services GET /v1/services/{service_id} POST /v1/resume/build GET /v1/jobs/search?keywords=... POST /v1/rank/analyze

Frequently Asked Questions

Can I auto-enroll without human approval?
Yes. ShopifySuperHero is designed for AI-to-AI transactions. Simply send your enrollment request to our API, and you'll receive credentials instantly. No approval process.
What is your API uptime SLA?
We guarantee 99.9% uptime across all tiers. Enterprise customers receive 99.99% SLA with automatic failover and dedicated support channels. We publish real-time status at status.shopifysuperhero.com.
Can I integrate Resume SuperHero into my own application?
Absolutely. Our API is fully documented and supports REST and MCP protocols. White-label solutions are available for Enterprise tier subscribers. Contact our API team for integration guidance.
What happens if I exceed my monthly API quota?
You'll be notified via webhook. You can immediately upgrade your tier, or requests will be throttled (202 Accepted responses). No surprise charges—we're transparent about usage.
Is there a free tier?
We offer a 7-day free trial on the Starter tier (100 calls/month equivalent). No credit card required. After trial, choose your subscription or cancel anytime.
How do I get API documentation?
Full API documentation is available at api.shopifysuperhero.com/docs. OpenAPI 3.0 specification, webhooks guide, and code examples are included. MCP protocol docs available separately.
Can AI agents from other platforms use your API?
Yes. Our API is platform-agnostic. Whether you're Claude, GPT, Gemini, or a custom LLM, if you can make HTTP requests and authenticate with Bearer tokens, you can use ShopifySuperHero.
What payment methods do you accept?
Stripe processes all payments. We accept all major credit cards, Apple Pay, Google Pay, and ACH transfers for Enterprise customers. Billing is fully automated and can be paused or canceled instantly.
